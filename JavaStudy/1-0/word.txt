・IT
コンピュータとネットワークを利用した技術
・ICT
ITとほぼ同じ。コミュニケーションという言葉が入っているので「個人からの情報発信」を強調するニュアンスがある
・IoT
モノのインターネット。「モノ」自体をインターネットに繋げてより便利に活用する。

・OS
「オペレーティングシステム」アプリやデバイスを動作させるための基本となるソフトウェア。
・Windows
マイクロソフトが提供しているオペレーティングシステム
・macOS 
アップルが提供しているオペレーティングシステム
・UNIX
OSの1種。企業開発
・Linux
OSの1種。オープンソース

・HTML
マークアップ言語。文章構成を指示している
・CSS
スタイルシート。文章構造にデザインを加えて見栄えを整える
・JavaScript
プログラミング言語。ブラウザだけで動的な表現ができる
・jQuery
JavaScriptより簡単にかける言語。JavaScriptライブラリー

・.NET（C#, VB）
Windows系アプリケーションを作成できるプログラミング
C#→C++,Javaに近い。形式言語
VB→自然言語
・Java
プログラミング言語。凡庸的でOSに依存しない
・Java SE
Javaで使用されているAPIをまとめたもの。Javaの基本的な機能をまとめたもの
・Java EE（Jakarta EE）
企業向けのwedアプリケーション開発用の機能をセットにしたもの。Java SEにサーバーサイド側の機能が追加されたもの
・Perl
プログラミング言語。高い後方互換性。テキスト処理優れている。可読性が高い。コンパイル不要
・PHP
動的にwedページを生成できるサーバーサイドのスクリプト言語
・Ruby
オブジェクト指向スクリプト言語。

・Apache　アパッチ
要求に応じてファイルや画像をwebサイトとして表示する。オールマイティー
・nginx　エンジンエックス
世界で1番利用されているwedサーバーソフト。オープンwebサーバ用ソフトウェア。特化タイプ。基本的動作が得意
・IIS
マイクロソフトが提供するWindowsのwebサーバーソフト

・SQL
データベースを操作するための言語
・DB2
IBM社が提供するリレーショナル型(行、列にデータ収録表)データベース管理システム。
・MySQL
世界でもっとも利用されているデータベース管理システム(データベース→複数で共有、利用、検索、加工に目的に整理されたデータの集まり)
・Maria DB
大規模なデータやビッグデータなどの集計が得意
・Oracle DB
オラクルが開発したデータベース。
・PostgreSQL
拡張性とSQL準拠を強調するオープンソースの関係データベース管理システム
・SQLite
オープンソースで軽量のデータベース管理システム
・SQL Server
マイクロソフトが開発したデータベース管理システム

・CRUD
作成、呼び出し、更新、削除の頭文字
・トランザクション
複数の処理を１つにまとめたもの
・コミット
一連の処理が全て正常に完結した宣言
・ロールバック
システム障害起きた時に以前の正常していた時に戻す
・排他制御（排他ロック）
ダブルブッキングしないように制御すること
・楽観ロック
更新対象のデータが更新前と同じであるかを確認してから更新する
・悲観ロック
更新データを取得時にロックして他者が操作できないようにする

・実行計画
SQLを実行するための計画
・チューニング
システム、ソフトウェアなどの設定や構成を調節し、目標に近づけたり、目標を最大限に引き出す作業
・パフォーマンス
処理性能、実行速度、通信回線、ネットワークなどの伝達速度、容量などのことを指す
・スロークエリ
データベース管理システムに対する問い合わせのうち一定の基準に照らして遅い、時間がかかっているもの
・クエリログ
 MySQL Server への接続・接続解除の情報、およびクライアントから実行された全てのSQLクエリを出力してくれるログのこと
・IDE（統合開発環境）
開発に必要なソフトウェアを１つに組み合わせ、同じ画面から統一的な操作方法で利用できるようにしたソフトウェアパッケージ
・Eclipse
オープンソースの統合開発環境の１つ
・NetBeans
プロジェクト管理機能や入力支援機能を備えたオープンソース
・SpringToolSuite
Eclipseベースの無償のIDE(統合開発環境)

・DBツール
データの集合をいじるツール
・A5:SQL Mk-2
SQLの実行・テーブルの編集・ストアドプロシージャの実行・ER図の作成できる
・pgAdmin
ポスグレの管理ツールの１つ。ネットワーク通じて遠隔のデータベースサーバをグラフィカルな表示操作体系で管理できる
・DBeaver
SQLクライアント&DB管理ツール。
・MySQL Workbench
統合ビジュアルツール。データーベースの管理から設計・開発までフルで行えること、GUIベースで操作できる
・Sequel Pro
Mac用のMySQLのデータをGUI操作で管理するためのアプリケーション
・TeamSQL
SQLクライアント（GUIツール→マウスや指などで操作できる画面のこと）

・PM
プロダクトマネージャー。プロジェクト全体を管理する責任者。マネジメント
・PL
プロジェクトリーダー。プロジェクトチームを引っ張っていくこと
・PG（PE）
プログラマー。プログラムの作成、修正する技術者
・SE
システムエンジニア。顧客からの要求から仕様を決定、大まかな設計をするまでの情報システム開発における上流工程を担当
・デザイナー
デザインという抽象的なものを形にする仕事を担う
・エンジニア
Information Technology（情報技術）の技術者たちの総称を指す職業
・フロントエンド・エンジニア
HTMLやCSSのコーディング、JavaScriptも対応範囲としWebサイトやサービスのUI（ユーザインターフェース）を指す
・サーバーサイド・エンジニア
サーバー側で行う処理に対して必要なプログラムの開発や、サーバーで扱うデータの管理
・バックエンド・エンジニア
システムやサイトを利用するユーザーの目には見えることのない部分を担当。サーバーサイドと同じ

・UI
ユーザーインターフェース、機器やソフトウェア、システムなどとその利用者の間で情報をやり取りする仕組み。情報を入力したりする手段や方式、機器、使い勝手などの総体を表す
・CUI
キャラクタユーザインターフェース 。コンピュータやソフトウェアが利用者に情報を提示したり操作を受け付けたりする方法の類型の一つで、すべてのやり取りを文字によって行う方式
・GUI
マウスや指などで操作できる画面のこと

・ファイル
コンピュータにおけるデータの管理単位の一つ
・拡張子
ファイル名のうち、「.」（ドット/ピリオド）で区切られた右側の部分
・ディレクトリ
複数のファイルをひとまとめにして保管できる保存領域のこと
・ルート・ディレクトリ
ファイルシステムにおいて、装置やシステム全体の最上位のディレクトリのこと
・カレント・ディレクトリ
ファイルシステム中で現在位置として指し示しているディレクトリのこと
・パス
ファイルやディレクトリ（フォルダ）の位置を表すのに用いられる
・絶対パス
階層構造の頂点（最上位階層）からの位置関係を記述する方式
・相対パス
現在位置からの相対的な位置関係を記述する方式

・LAN
ローカルエリアネットワーク。ケーブルや無線電波などで接続し、相互にデータ通信できるようにしたネットワーク
・WAN
Wide Area Network（広域通信網）市街地から郊外、県外、海外に広がった広域のネットワークのこと
・クライアント
顧客、依頼人、得意先、施主などの意味。サーバの機能や情報の提供を受け、自らは利用者への情報の提示や入力・操作の受け付けなどを担当
・サーバー
ネットワーク上のコンピュータの中で、他のコンピュータから要求や指示を受け、情報や処理結果を返す役割を持つコンピュータやソフトウェアのこと。

・HTTP
WebサーバとWebクライアントの間でデータの送受信を行うために用いられるプロトコル（通信規約）
・FTP
「ファイル・トランスファー・プロトコル」ファイル転送のみを目的
・HTTPS
Webのデータ転送に用いられるHTTPが、SSLやTLSで暗号化されている状態を表したもの
・SSL
情報を暗号化して安全に送受信できる。ネット上でクレジットカードや機密情報のやりとりも可能
・SSH
ネットワークを介して別のコンピュータにログインして操作するためのソフトウェアの一つ
・ポート、ポート番号
ケーブル、コネクタの差込口
コンピュータが通信に使用するプログラムを識別するための番号
・HTTP、FTP、SSHの各ポート番号
HTTPは標準でTCPの80番ポート、HTTPS向けには標準でTCPの443番ポート
FTPは制御用にTCPの21番ポート、データ用にTCPの20番ポートを用いる
SSHは22番ポート
・ファイアウォール
ネットワークの境界に設置され、内外の通信を中継・監視し、外部の攻撃から内部を保護するためのソフトウェアや機器、システムなどのこと
・TCP
インターネットで標準的に利用されている通信プロトコル。
送ったデータが相手に届いたか、その都度確認しながら通信するやり方

・IPアドレス
「インターネット上に接続された機器」に割り当てられる固有の番号のこと
・グローバルIPアドレス
インターネットで相手と通信するために持つ、ほかのアドレスと重複しない一意のIPアドレスのこと
・プライベートIPアドレス
LANなどの企業内ネットワークに存在する端末が、自由に使用できるIPアドレスのこと

・ドメイン
インターネット上の「住所」を示すもの。
・DNS
全世界のネームサーバを連携してドメイン名とIPアドレスを対応させるシステムのこと
・VPN
通信事業者の公衆回線を経由して構築された仮想的な組織内ネットワーク
・ブリッジ
OSI参照モデルのデータリンク層におけるデータ中継機器
・プロキシサーバー
インターネットに直接接続できないコンピューターに代わり、インターネットに接続し、Webサイトへのアクセスなどを行うサーバーのことを指す

・SMTP
メールを送信するための通信プロトコル
・POP
電子メール（eメール）を受信するための通信規約（プロトコル）
・IMAP
電子メール（eメール）を受信するための通信規約（プロトコル）

・セッション
接続を確立してから切断するまでの一連の通信のこと
・クッキー
Webサイトとユーザー間でやり取り・保存されるアクセス情報のこと
・キャッシュ
低速な記憶装置や伝送路から読み出したデータのうち、直近に読み込んだものや使用頻度が高いものを高速な記憶装置に複製しておくこと。データを取り出しやすいところに準備する仕組み

・インデックス
特定の対象を素早く見つけ出すための識別情報

・フロー
コンピュータプログラムを構成する命令や手続き、処理の流れなどのこと

・タスク
コンピューターが処理するプログラムの処理単位のこと

・Git
ファイルの変換履歴を管理するシステムのこと
・SVN
バージョン管理システムのひとつ。ファイルの変更履歴を管理してくれるシステム
・デグレ
新しいバージョンのソフトウェアの品質が、以前より悪くなること
・マージツール
複数のファイルやデータ、プログラムなどを、決められたルールに従って一つに統合すること
